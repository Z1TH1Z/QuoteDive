# Quotes App — Philosophy Learning Platform with Native Widget and Gamification

A production-ready Flutter application that delivers daily philosophy quotes through a native home screen widget, combined with structured learning, gamification, and offline-first architecture.

Built end-to-end with scalable architecture, local persistence, and real-time widget synchronization.

---

## Highlights

• Built complete mobile product from scratch to production  
• Implemented native Android home screen widget integrated with Flutter state  
• Designed offline-first architecture using SQLite  
• Engineered full gamification system (points, streaks, levels, badges)  
• Implemented structured learning with deep dives and quizzes  
• Created scalable modular architecture using clean separation of concerns  
• Achieved real-time widget updates using platform integration  
• Production-ready performance and architecture  

---

## Demo Overview

Core flow:

1. User sees daily quote directly on home screen widget  
2. Opens app and explores structured deep dive  
3. Earns points, unlocks badges, increases streak  
4. Widget updates instantly to reflect progress  
5. All functionality works fully offline  

Demonstrates:

• Native platform integration  
• Persistent state management  
• Real-time UI synchronization  
• Production-level mobile architecture
